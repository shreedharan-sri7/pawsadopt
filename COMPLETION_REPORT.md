## 🚀 How To Start

### Quickest Way (Recommended)
```bash
# Windows
.\start.bat

# Mac/Linux
bash start.sh
```

### Manual Way
```bash
# Terminal 1 - Backend
cd backend
mvn clean install
mvn spring-boot:run

# Terminal 2 - Frontend
cd frontend
npm install
npm start
```

### Then Open
- Frontend: http://localhost:3000
- Backend API: http://localhost:8080/api
